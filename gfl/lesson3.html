üåå 1. Primary Control Objectives

  • Active Power Injection: Deliver real power from the DC source to the grid
  • Reactive Power Control: Adjust reactive power for voltage support or compliance
  • Power Factor Regulation: Maintain a desired power factor (often unity)
  • Current Limitation: Protect inverter components during disturbances

üìä 2. Voltage Reference Tracking

GFL inverters measure grid voltage and regulate their current output to track this reference. The PLL plays a key role in identifying the voltage phase angle. Current commands are generated in the dq reference frame and translated back to abc phase signals for PWM generation.

üîÑ 3. Control Architecture

  • Outer Power Control Loop: Sets current references based on P/Q commands
  • Inner Current Control Loop: Ensures injected current matches the reference
  • PI Controllers: Commonly used to minimize error in both loops

üîß 4. Additional Features

  • Anti-islanding protection to avoid unintentional islanded operation
  • Voltage ride-through to remain online during sags
  • Dynamic reactive support depending on grid codes

‚úÖ Summary

  • GFL control focuses on current injection aligned with grid voltage
  • Current control is performed in a dq synchronous frame
  • Power factor, voltage tracking, and anti-islanding are key features
