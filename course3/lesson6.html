1. Dispatching Inverter Resources

Inverter-based resources can be dispatched based on power needs, grid strength, and system objectives. GFM inverters provide support services like frequency regulation, while GFL inverters focus on following active and reactive power setpoints.

2. Dual Control in Dispatch

The duality theory enables a control structure where both GFM and GFL inverters can participate in grid support, each with clearly defined roles. This allows dispatch systems to assign tasks such as voltage control to GFM units and tracking to GFL units based on grid conditions.

3. Role of Coordination and Setpoints

  • GFM Setpoints: Typically set power droop curves, frequency targets, or reactive support policies
  • GFL Setpoints: Receive real-time current references based on system demand and inverter limits
  • Coordination: Dispatch algorithms must respect system constraints, latency, and response times
